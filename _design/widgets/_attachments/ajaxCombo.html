ajaxCombo widget

This widget displays a combo box to select a country.

The options are retrieved from the database via an Ajax call during the initialization phase of the widget.

In order to use this widget, first you have to upload the countries.json file intro the database.

Also a view is required for the Ajax call, to return the list of countries.

You can define the name of the view, from which the option list can be filled up in the attribute list of the widget:
$("#ajaxCombo").evently( "ajaxCombo", app, { "viewName" : "countries" } );

If you want the widget to automatically make the appropriate option to be selected, set the 'key' into the 'current' attribute.
For example: current="VUT".
